# ⚛ Quantum Communication Dashboard

An interactive Quantum Communication Dashboard built with **Flask**, **Qiskit**, and **Python** to demonstrate fundamental quantum computing concepts through an engaging web interface.

## 🌌 Project Overview

This project explores key concepts in Quantum Communication through interactive simulations and visualizations.

Current modules include:

- ⚛ Bell State Generation
- 🔗 Quantum Entanglement
- 🚀 Quantum Teleportation
- 📊 Quantum Circuit Visualization
- 📈 Measurement Results

The long-term goal is to evolve this into a modern, immersive Quantum Communication platform with real-time simulations, premium UI/UX, and IBM Quantum hardware integration.

## 🚀 Installation

Clone the repository

```bash
git clone https://github.com/yashas396/Quantum-Communication-Dashboard.git
```

Navigate into the project

```bash
cd Quantum-Communication-Dashboard
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the application

```bash
python app.py
```

Open your browser

```
http://127.0.0.1:5000
```
